Dashboard Ninja Advance

Ksolves Dashboard Ninja Advance opens up the limitless possibilities for you to present your complex data in form of Charts or list view on Dashboards with awesome query feature of Dashboard Ninja. This app facilitates you to view the Dashboard Items in full Screen with Slider

Key Features

  • TV Mode - Items presentation with slide show view
  • Query feature - Now user can write Sql query by its own to fetch records from Data Base.
  • Layouts - New layouts added for List view

User Guide

1. TV Dashboard Mode

You can go into the Tv mode of Dashboard by clicking on Play button

You can set the slide Interval from Dashboard Configuration page

You can pause the item slider by hovering on it.

2. Creating List View or Chart with Sql Query

You can create the List view with Query by Selecting query as Data Calculation type in Item form View

Go to the Query tab in the form view

Enter your own query for fetching the Data

You can also draw the chart with query feature
